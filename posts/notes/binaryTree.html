Binary Tree

A tree with maximum 2 nodes is called "Binary Tree". It can be traversed in depth first way. Regardless of level of tree, it considers going through depth first. It searches from leaves and can be implemented with recursive codes. There are 3 ways to do it, in-order Traversal, pre-order Traversal, post-order Traversal.

Because every node needs to be visited at the worst case, time complexity of all traversals is O(n).

- In-order Traversal (Left-Root-Right)

A node on the tree can be divided by left subtree and right subtree. The logic is that after searching on left subtree, visit the root and search right subtree. The pattern goes over again.

With Recursion

As finding the last and left node with no left child node by recalling inorder(node.left, stack), the node is inserted in stack. Then, it goes to right subtree to find the last and left node with no left child node by calling inorder(node.right, stack). 


                                        // Recursive
                                        // Time complexity: O(n)
                                        // Space: O(n)

                                        var inorderTraversal = function(root) {
                                            let stack = [];
                                            inorder(root, stack);
                                            return stack;
                                        };

                                        function inorder(node, stack){
                                            if(!node) return stack;
                                            inorder(node.left, stack);
                                            stack.push(node.val);
                                            inorder(node.right, stack);
                                            return stack;
                                        }

With Iteration

There is a stack to track the node and result for returning the result. When the last and left node does not have left child node, it is popped from the stack and inserted in result. Then it goes to right tree to find the last and left node with no left child node. To make sure to go through every node, stack needs to be checked whether it is empty or not. 


                                        // Iteration
                                        // Time complexity: O(n)
                                        // Space: O(n)

                                        var inorderTraversal = function(root) {
                                            let stack = [];
                                            let result = [];
                                            let node = root
                                            while(node || stack.length>0){
                                                while(node){
                                                    stack.push(node);
                                                    node = node.left;
                                                }
                                                node = stack.pop();
                                                result.push(node.val);
                                                node = node.right;
                                            }
                                            return result;
                                        };
