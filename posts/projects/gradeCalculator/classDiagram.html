Class Diagram

Grade calculator consists of 4 classes in the diagram, which are Grade, Requirement, Weighted, and Not-Equal-Weighted.

- Class Description

Grade

Grade class contains 7 attributes, which are course code for the subject, number, type, title, score, full mark, and counter for best mark.

courseCode : Up to 10 characters and need to uniquely defined.
no : Ordinal number of an assessment.
type : 1 character with W, Q, A, or T. 'W' for weekly task, 'Q' for Quiz, 'A' for Assignment, and 'T' for Test.
score : The mark the student earned.
fullMark : Maximum mark that student can earn.

Requirement

It has 4 attributes, which are course code for the subject, overall weight to pass, and 2 policies.

courseCode : Up to 10 characters and need to uniquely defined.
overall : Overall mark to pass the course.
policy1, 2 : Extra conditions that need to be satisfied to pass the course

Weighted

It has 7 attributes, which are course for the subject, type of assessment, whether it is equal weighted assessment, name of assessment, total weight for it, the number of assessment that counts for final grade, and max for the maximum number of assessment for the term.

course : Up to 10 characters and need to uniquely defined.
type : 1 character with W, Q, A, or T. 'W' for weekly task, 'Q' for Quiz, 'A' for Assignment, and 'T' for Test.
equal : Whether the assessment type is evaluated equally or not.
assessName : Name of the test.
totalWeight : Total weight of the assessment type.
count : The number of assessments that counts for final grade.
max : Maximum number of assessment.

Not-Equal-Weighted

It has 1 attributes, each weight to store not equal weighted assessment. Other attributes are inherited by Weighted class.

eachWeight : When type of assessment has different weight on each assessment, each weight will be stored separately.

- Association

Grade class does not have an association with any other class.
Requirement of a course can have many weighted class with different type of assessment.
When assessment has different weight on each assessment, it will have a notEqualWeighted that inherits Weight class.
