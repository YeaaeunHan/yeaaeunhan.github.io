Object Level Sequence Diagrams

Grade calculator helps students to insert, update, and delete scores. Acceptable criteria are 1) the ability to record grades 2) the ability to query those grades to calculate grades.

The program adds a grade with number, type, title, score, full mark. It finds the exact grade by number and type.

- Insert Grade

Use Case Description
Use Case Name Insert a grade
Triggering Event A student requests to add a grade
Brief Description Allows the student to record a new grade
Actors Student
Pre-conditions Student has opened the main menu.
Post Conditions Grade is saved to the database.
Flow of Activities Actor System
Requests to insert grade. Prompts for number, type, title, current score, and full mark.
Enter number, type, title, current score, and full mark. Verifies number, type, title, current score, and full mark has been entered.
Displays number, type, title, current score, and full mark.
Requests to save. Saves a grade.
Exception Conditions Student chooses to cancel adding the grade.

- Update Grade

Use Case Description
Use Case Name Change a grade description
Triggering Event A student requests to update the grade description
Brief Description Allows the student to change a grade description
Actors Student
Pre-conditions Student has opened the main menu.
Post Conditions Grade is changed and able to query.
Flow of Activities Actor System
Requests to change grade description. Prompts for number, type.
Enter number, type. Verifies number, type has been entered.
Retrieves the grade description and prompts for update.
Update number, type, title, current score, and full mark. Displays number, type has been entered.
Prompts for update.
Requests to save. Saves the grade and returns to the main menu.
Exception Conditions Student chooses to cancel updating the grade.

- Delete Grade

Use Case Description
Use Case Name Delete a grade
Triggering Event A student requests to delete a grade
Brief Description Allows the student to delete an existing grade
Actors Student
Pre-conditions Student has opened the main menu.
Post Conditions The grade is deleted.
Flow of Activities Actor System
Requests to delete grade. Prompts for number, type.
Enter number, type. Verifies number, type has been entered.
Retrieves the grade description and prompts for delete.
Chooses to delete. Deletes the grade and returns to the main menu.
Exception Conditions Student chooses to cancel deleting the grade.
